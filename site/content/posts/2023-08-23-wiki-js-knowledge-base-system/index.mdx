## 時空背景
原本想解決部門內資料共用知識，但其他部門有這個需求，因此從公司面思考知識共享的議題，原本在使用的 OneNote 也被很多工程師，反應不太好用，我自己也覺得，因此想看看有沒有其他的解決方案，後來發現了好用的 Wiki.JS。

## 費用
Wiki.Js 支援多人共同維護，不需額外費用，開源程式碼，且免費，並支援 Docker 佈署

### Docker + Mysql 指令
