## 談談英文教育?
之前聽到有個笑話，台灣人學英文的時間省下來可以學一個技能，

從小到大我們的英文教育，將近10年，但成效好像沒這麼好，我想這個是大多人心中的痛，我們缺乏使用的環境。  

## 英文真的值得投資嗎?
如果我必須工作到65歲甚至更老，如果花2~3個月的密集學習，就可以練到一定的層度，接著就可以靠自己學習，對我來說這筆投資就很值得，因為未來還可以用20~30年。

## 我的背景
我其實是鄉下出身，在出社會前不知道為何學英文，該怎麼學英文，除了考試，學了什麼機會可以用? 其實這些在成長的過程中，我並不知道。

半開玩笑的說，要在鄉下要遇到外國人，可能比被石頭砸到機率還低。  

我擁有10年軟體開發經驗，服務過在台灣、大陸、新加坡及美國公司，但讀寫OK，聽說就不太行。  

曾經去過地球村、也試過 AI 家教，但那對我真的沒太大的幫助，因為我並沒有用到生活中。

## 起心動念
從[10年前一場面試說起](https://blog.markkulab.net/career-boundaries/)，資策會畢業前，接到美商UPS邀請我去面試，我在我同學勸說下去面試，我用著很吃力的用我的破英文，念著UPS 的 policy 及自我介紹，當時也幸運的拿到offer，但因為一些小插曲，並沒去報到。  

幾年前公司的會議，主持會議的是前老闆，靠著是前老闆 cover，又逃掉了英文口說。  

2年前臨時被安排的美國出差，靠著同事幫忙及硬拼單詞及畫流程圖，完成了一第次的需求訪談，並把需求規格帶回來台灣。  

接連著，美國同事來台灣及一些外部廠商的英文會議，發現自己使不上力，事後都得靠會後發信總結來確認。  

這些種種，讓我開始意識到英文重要性，讓我永遠記得那種有口有卻無法表達的感覺。  

2024 年終於意識到英文是10年前的問題，只是繞過去，但問題依舊還是在那。  
   
## 必須要有的體認
學習一個技能或是一個語言，不會是看本書就能學會，或幾花幾百個小就能學會的，就必須不斷的使用，不會短期就能學會，那可能也沒這個價值學，學習一個專業是要長期累積的，就和鐵人三項，你不會練個三個月就去比超級鐵人，如果有，也是原本的條件就很好。

## 實際行動

雖然2023年我的工作很忙，在兵荒馬亂中度過，但2024 年起下定決定決心把英文學好，

我開始上了菲律賓全英的線上家教 ( 60 堂 * 50 min ) 

剛開始上全英文家教時，每堂課很辛苦，都覺得鴨子聽雷，每堂課前都要準備個1~2小時，

但直到後20堂後，我發現，我漸漸能夠很直覺得把英文用在生活中，或是用英文去思考。

2024 年 7月在德國網站專案順利上線後，  

利用特休及補班去了語言學校 ( 2週 )，同時也交了很多好朋友，才發現有更快速學語言的方法，不外乎就是用，心理萌生去更長期語言學校。

## 從基礎開始，我做了那些的努力
因為平常要工作，要用利閒的時間讀書，英文很多年沒用，而且腦中常常有很多工作的事，一開始學習的進度並不好，也很挫折。

從 ESL Level 2 >　Level 4
剛開始我買了 Hi 家教的線上課程(60 堂 x 50 min ) 課程，我花了不少時間在建立信心，終於找到一位老師願意聽你說什麼，慢慢修你說的東西，可以自然的表達你生活遇到的事情。  

接著再利用特休和補休去菲律賓 [CBOA 上了兩週的課](https://blog.markkulab.net/cebu-blue-ocean-academy/)  (8天 * 8堂課  = 64 x 45 min )

這期間，累積一共 130 堂的英文課。  

ESL Level 4
在CBOA 我得知了， [11 talk ](https://11talk.net/)這個平台，為了讓我的英文學習成效更好，在去Pines 前我移轉到了 11 talk (Pines 的線上平台)，  

ESL Level 4~ Level 7   
我打算待10週在碧瑤的Pines語言學校，給自己目標是從Level 4 到 Level 7

## 學習英文目的目標
1. **提升跨國溝通能力**  
   在工作和生活中，能自信地與不同國籍的人交流。不再害怕與外國人溝通，例如，這幾個月我曾幫助過幾位外國人，這是我過去無法想像的突破。  

2. **拓展思維與自我探索**  
   英語學習過程中，經常面對開放性問題，沒有標準答案，讓我開始思考教育中從未涉及的話題，例如：  
   - 你的遺願清單是什麼？  
   - 如果有機會，你會嘗試什麼新事物？  
   - 你如何定義成功？  
   這些問題讓我更加了解自己的目標與價值觀。  

3. **了解不同文化**  
   透過語言學習，我得以深入認識其他國家的文化背景，從而擴展世界觀。  

4. **結交新朋友**  
   去年參加語言學校後，我認識了許多外國朋友，尤其是來自日本的朋友，我們透過英語交流，建立了珍貴的友誼。  

5. **工作之餘的休息與調整**  
   由於工作壓力，我過去忽視了健康，曾因高血壓兩次進入急診室，藉由學習英語，作為休息與調整心態的一部分，並透過持續用藥控制健康。  

6. **拓展職業與人生機會**  
   英語能力的提升，能帶來更多的職場及任何新的可能性。

## 為什麼選擇 Pines Academy 語言學校
去年去過了 [CBOA 語言學校](https://blog.markkulab.net/cebu-blue-ocean-academy/)，環境真的很棒，也很好玩，相較之下姐妹校的Pines，是， 環境雖然沒這麼好，碧瑤整體讀書風氣會普遍比宿霧好，可以讓我更專心在學習，華人比例也會較低，但因為他和 CBOA 是姐妹校，Pines也有經營線上課 11 talk，我試過一堂後，覺得資師及教學系統蠻有結構的，最後就選擇了Pines Academy。 

## 為什麼選擇 Insensitive ESL
在有了上次CBOA 的語言學校經驗，所以不希望一整天都在上課，這樣會沒時間複習，成效反而不好，對我最重要最重要的是能夠順暢溝通及表達，把底子打好，並且加強口說和聽力，像是多益分數，我可以回台灣自己再慢慢練刷題。

## 補充 - 附上英文等級的標準的對照
CEFR 是 Common European Framework of Reference for Languages 的縮寫，中文翻譯為 歐洲語言共同參考框架，它是一個用來評估和描述語言能力的國際標準，適用於不同的語言學習者。CEFR 將語言能力分為六個主要等級，從初學者到熟練使用者，分別是：

![Standard Comparison of English Proficiency Levels](english-level.jpg)

| CEFR Level | ESL Level | 描述說明                                                                                             |
|------------|---------------------|--------------------------------------------------------------------------------------------------|
| C1         | 10                  | 能夠理解廣泛的主題，更長的文章並能識別隱含的含義；可以在任何社交，學術或專業環境中流暢、有效的表達觀點；甚至可以在復雜的主題上能夠清晰、有結構且合理的說明，並有邏輯的表達。 |
| B2         | 9                   | 可以參與各種對話；可以閱讀和理解複雜的內容文字，例如報紙文章、雜誌或期刊；可以理解他人的觀點，並就某個特定問題發表的意見，即使是母語人士也可以輕鬆面對。               |
|            | 8                   | 能夠理解對話或討論的重點，只要該主題是熟悉的或所在的興趣和或專業領域內；可以重點出必要的訊息並提出問題，以進一步的辯論、澄清或說明；可以編寫詳細的內容，闡述有關主題的觀點。      |
| B1         | 7                   | 可以詳細、有把握地解釋在特定情況下出現的問題，並可以提出如何處理、建議；可以理解對自己有興趣或擅長的狀況問題，並適當地回應其他人的後續問題。                       |
|            | 6                   | 可以理解大多數與熟悉事物有關的想法，並且可以在旅行時處理大多數情況；可以描述經歷、事件、夢想、希望和抱負，並簡單說明觀點和計劃；可以維持有關個人興趣的具體話題的社交對話。          |
| A2         | 5                   | 可以交換意見和情緒，表達不同意見，並使用簡單的表達方式比較人與事；可以在社區、工作場所和學校的日常情況下使用多種表達方式與他人互動。                         |
|            | 4                   | 可以為熟悉的特定地方提供簡單的指導；可以藉由照片、地圖和一系列記憶的表達方式簡短地談論的鄰居、學校和工作；可以使用基本單詞撰寫邀請函和其他簡單的個人信件。           |
| A1         | 3                   | 可以問和回答有關熟悉主題的簡單問題；可以使用基本短語提出；只要有書面說明，就可以談論個人經歷。                                          |
|            | 2                   | 可以使用有限的單字量和其他表達方式來理解關於熟悉的話題的簡短對話，例如興趣愛好、食物、學校和工作。                                   |
|            | 1                   | 可以閱讀和理解非常簡單的說明和書面通知；可以詢問和回答有關時間、日期和價格的簡單問題；可以使用記憶式句型表達自己，可以抓住關鍵點並遵循基本指示，前提是問題以非常緩慢和清晰的方式傳達。 |
| Pre-A1     | 1P                  | 可以使用日常熟悉的單詞，表達記憶的表情和季節性的問候；可以識別簡單的書面文字，也可以通過複製產生相同的文字；可以傳達有關自己的簡單信息，例如姓名，年齡，地址和原籍國。       |
